
    Response:
    ```typescript
    { success: boolean; invoice?: InvoiceHistory; error?: string }
    ```

    Flow:
    1. Validate request body (customerId required, billingMonth in YYYY-MM format)
    2. Fetch customer from MongoDB (INV-03)

    7. Call AutoCount API via autocountClient.createInvoice (INV-06) - handle mock mode
    8. Save invoice record to MongoDB via invoiceRepository.insertInvoice with status (INV-07)
    9. Return success/error response

    Mock mode (INV-09):

    - Do NOT call real AutoCount API in mock mode
